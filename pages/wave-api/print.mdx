# WavePrint Function

The `WavePrint` function is a utility for printing formatted messages to the console.
You can import it from `wave-shell` or grab it from the `running options`

import { Callout } from 'nextra/components'
 
<Callout>
  If you are using prompt from the running options, the context will be automatically set to the name of your cli. You can customize it inside `/bin/cli-name`
</Callout>

### Usage from running options

```typescript copy
import { WaveCommand } from 'wave-shell'

export default {
  run: async (options) => {
    const { print } = options;
    print.success('Hello World'); //output: [Wave] Hello World

  }
} as WaveCommand

```

### Usage from wave-shell

```typescript copy
import { WavePrint } from 'wave-shell'

const print = WavePrint('MyCli');

print.success('Hello World'); //output: [MyCli] Hello World
```

## Parameters

* Context: the context of the message. It will be used to format the message like this: `[Context] Message`

## Methods

### success

Prints a success message to the console

```typescript copy
print.success('Message');
```
---

### error

Prints an error message to the console

```typescript copy
print.error('Message');
```

---

### info 

Prints an info message to the console

```typescript copy
print.info('Message');
```

---

### table 

Prints a table to the console

```typescript copy
print.table(['John Doe', 32], {
  header: ['Name', 'Age'],
})
```

---

### spaceLine

Prints a space line to the console

```typescript copy
print.spaceLine();
```

---

### clearLastLines

Clears the last lines from the console

```typescript copy
print.clearLastLines(2);
```
