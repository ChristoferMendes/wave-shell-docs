## Prompt

This module provides a set of functions to prompt the user for input in a CLI environment.

## Methods

### ask

This method takes a single string argument and returns a promise that resolves to the user's input.

```javascript copy
const result = await prompt.ask('What is your name?');
```

---
### confirm

This method takes a single string argument and returns a promise that resolves to a boolean value.

```javascript copy
const result = await prompt.confirm('Are you sure?');
```
---
### select

This method takes a single string argument and returns a promise that resolves to the user's selection.

```javascript copy
const result = await prompt.select('What is your favorite color?', ['red', 'green', 'blue']);
```
